#### Vue.js

- Super Simple Way to Add Global Method.
```javascript
// In your file that init the Vue app.
Vue.mixin({
    methods: {
        myGlobalMethod: function (myParam) {
            console.log(myParam);
        },
    }
});

```

#### Formulas

- How did I measure the performance increase?
```javascript
((old - new) / old) * 100%
= ((20 sec - 5 sec) / 20 sec) * 100%
= 75 % (performance increase) 
```
